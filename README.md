# Cafe Attendance Admin (cafe-employment)

A premium web application for managing cafe staff attendance records with AI-powered OCR.

## Features
- **OCR Data Extraction**: Automatically extracts date and time from dot-printed time cards.
- **Real-time Dashboard**: Visualizes work hours and labor statistics.
- **Staff Management**: Consolidated view of employee work patterns.

## Tech Stack
- React + Vite
- Tesseract.js
- Lucide Icons
- Framer Motion
